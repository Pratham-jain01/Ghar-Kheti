
# Ghar-Kheti Analytics Dashboard (React + Vite)

This project is a modern React dashboard powered by Vite and styled with Tailwind CSS.

## Getting Started

1. **Install dependencies:**
	```bash
	npm install
	```
2. **Start the development server:**
	```bash
	npm run dev
	```
3. **Build for production:**
	```bash
	npm run build
	```

## Project Structure

- `src/` — React source code (main.jsx, App.jsx, index.css)
- `index.html` — Main HTML entry
- `tailwind.config.js`, `postcss.config.js` — Tailwind CSS setup
- `.env` — Environment variables (if needed)

## Features

- React 19 + Vite for fast development
- Tailwind CSS for styling
- ESLint for code quality

---

> **Note:** This project previously included a Streamlit dashboard. All analytics are now intended to be implemented in React.
